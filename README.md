# Global COVID-19 Data Analysis using Power BI

## Project Overview
This repository contains an **academic Power BI dashboard project** that analyzes global COVID-19 data from **January 2020 to May 2022**.  
The objective of the project is to explore pandemic trends, compare country-wise impact, and identify top and bottom performers using interactive data visualizations.

The dashboard is designed for **academic learning and data visualization practice**, focusing on clarity, interactivity, and analytical insights.

---

## Dashboard Structure
The Power BI report consists of **4 pages**, each serving a specific analytical purpose:

1. **Global COVID-19 Impact Summary**
   - Key metrics such as total cases, deaths, recoveries, and active cases
   - High-level overview of the global situation

2. **Global COVID-19 Trend Analysis**
   - Time-based analysis of total cases, daily new cases, and deaths
   - Drill-down capability from Year to Month
   - Interactive slicers for time and country filtering

3. **Country-Wise Comparison**
   - Comparison of COVID-19 impact across different countries
   - Analysis of case counts and trends by country

4. **Top & Bottom Performers**
   - Top 10 countries by total cases
   - Top 10 countries by death rate
   - Bottom 10 countries by total cases
   - Ranking-based insights using Power BI filters

---
